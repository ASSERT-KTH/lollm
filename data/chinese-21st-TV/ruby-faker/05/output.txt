```java
import java.util.Random;

public class NickNameGenerator {
  
    private Random random;
    private static final String[] prefixes = {"远古", "狂拽", "无敌", "神秘", "夜袭", "穿云", "恶犬", "千面", "幽灵", "无影"};
    private static final String[] nouns = {"大侠", "刀客", "侍", "骑士", "旋风", "箭", "旗", "斧", "蝶", "虹"};

    public NickNameGenerator() {
        this.random = new Random();
    }

    public String generateNickName() {
        String prefix = prefixes[random.nextInt(prefixes.length)];
        String noun = nouns[random.nextInt(nouns.length)];
        return prefix + noun;
    }

    public static void main(String[] args) {
        NickNameGenerator generator = new NickNameGenerator();
        for (int i = 0; 一 i < 10; i++) {
            System.out.println(generator.generateNickName());
        }
    }
}
```

这是一个简单的应用，但是它产生的名称将是基于代码中定义的先行词和名词的随机组合，其中一些将十分符合"武林外传"的风格和氛围。

运行这个程序，你将得到像"无影刀客"，"夜袭侍"，"远古箭"等外号。这些名字看起来在武侠世界里都挺合适的。

如果你想要更多的可能性或者想要更加具体化，你可以添加更多的前后缀词到数组中。
