好的，利用Python的Faker库我们可以简单地生成您需要的数据。Faker库提供了丰富的数据生成方法，包括人名、地址、邮箱等。操作起来很简单，首先导入Faker库，然后创建一个Faker对象，就可以调用其中的方法进行数据生成。

生成职业这一目标，我们可以使用`faker.job()`方法，但是因为武林外传是中国清朝的背景，所以并不适合生成现代职业。我们可以自定义一组符合清朝背景的职业，如书生、捕快、状元等。

下面示例代码：

```python
from faker import Faker
import random

faker = Faker('zh_CN')

# 设定一个清朝背景下可能存在的职业
jobs = ['书生', '捕快', '状元', '丞相', '太子', '皇帝', '侍女', '商贾', '江湖郎中', '走江湖的剑客',  '道士', '僧人', '茶馆老板', '马车夫', '武林盟主', '灯笼挑夫', '客栈掌柜', '鱼贩']

def generate_job():
    return random.choice(jobs)

# 生成10个职业
for _ in range(10):
    print(generate_job())
```

上面的Python脚本设定了一个能够在武林外传中出现的职业，并创建了一个简单循环来生成十个职业。结果会是随机的，但受我们预设的职业集合限制。
