The lyrics mention four personal names: නඩරාජා මල්ලියේ, සුමනසිරි මල්ලියේ, සෙල්ලම්මා නංගියේ, සෙනෙහෙලතා නංයේ. 

Here is a simple Ruby script that uses the faker gem to generate test data with these names:

```ruby
require 'faker'

def generate_test_data
  names = ["නඩරාජා මල්ලියේ", "සුමනසිරි මල්ලියේ", "සෙල්ලම්මා නංගියේ", "සෙනෙහෙලතා නංයේ"]
  
  10.times do
    name = names.sample
    email = Faker::Internet.email(name: name, separators: '.')
    phone = Faker::PhoneNumber.phone_number

    puts "Name: #{name}, Email: #{email}, Phone: #{phone}"
  end
end

generate_test_data
```

This script generates 10 sets of test data, each with a name randomly chosen from the list of names mentioned in the song, and a fake email and phone number generated by the faker library. The email is generated with the selected name as the username, using a dot as the separator.
